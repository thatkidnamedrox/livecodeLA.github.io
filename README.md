# livecodeLA.github.io

Landing page for livecodeLA. This site is a small accessible landing page that links to Discord and socials.

To preview locally:

```bash
# from the repository root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Assets:

- Upload your logo to [assets/logo.png](assets/logo.png).
- Add Hack font files to `assets/fonts/` (see [assets/README.md](assets/README.md)). The CSS will use them if present.

Edit `index.html` to change the Discord invite and social links.
